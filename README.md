## Development
To run this project:
### Run the console
```bash
./console.py
```
or (for windows)
```bash
python console.py
```

### Running Unit tests
```bash
python3 -m unittest discover tests
```

### Conventions
using [pycodestyle](https://pypi.org/project/pycodestyle/)

Test a sample file as:
```bash
pycodestyle console.py
```
